## Overview

Nezur is a Windows-oriented Roblox script executor made for running Lua scripts through a streamlined desktop interface and an integrated script hub. It is aimed at Windows 10 and Windows 11 users who want a simple executor setup with Lua injection, script organization, and fast access to widely used Roblox scripts.

The project also ships with an auto-update engine to help the executor track newer releases and shifting Roblox environments. Along with batch execution, a persistent script queue, multi-language UI options, and a built-in debugger, Nezur presents itself as a practical Windows tool for users who want a more direct script hub and executor experience.

Yes, the project profile is centered on Windows support.

**Will it still work after Roblox updates?**  
The included auto-update engine is intended to help the executor stay aligned with newer builds, but compatibility can still change over time.

**How is this different from other Roblox executors?**  
Nezur focuses on a lightweight UI, Lua execution, a 500+ script hub, batch execution, and a built-in debugger.

**Does using a script executor affect my account?**  
Any use of third-party tools can carry account-related risks. Review Roblox terms and decide what is appropriate for your setup.

**Are scripts stored locally?**  
Yes, the profile includes a persistent script queue, which suggests local retention of queued items between sessions.

**Can I update manually?**  
Yes, you can replace the local build with a newer version when available, or use the auto-update flow if enabled in your setup.
